Graph Functions

matrix_from_tables

Create a matrix and attributes from one or more dataframes.

Parameters:

  • tables: pandas.DataFrame or list of pandas.DataFrame. The data to be used to create the graph. If a list of dataframes is passed, each dataframe is used to create a separate graph.
  • relationships: list of lists. The relationship pairs to be used to create the graph. Types:
    - list of pairs: Each element of the list is a list of two elements, which are the names of the columns to be joined. All tables will be searched for these column names.
    - list of lists of pairs: the number of lists is equal to the number of tables, to specify relationship pairs for each table. Each sublist is a list of pairs of columns to link.
  • dynamic_col: str or list of str. The name of the column containing the time information. If a list of strings is passed, each string is the name of the column containing the time information for each dataframe in data. Default = None, meaning there is no temporal aspect.
  • weight_col: str or list of str. The name of the column containing edge weights. If a list of strings is passed, each string is the name of the column containing the weight information for each dataframe in data. Default = None, which means all the edges will have weight one.
  • join_token: str. The token used to join the relationship name and the ID name. Default ='::'

Returns:

  • A: scipy.sparse.csr_matrix. The matrix of the graph.
  • attributes: list of lists. The attributes of the data points. The first list contains the attributes of the data points that do not change over time. The second list contains the attributes of the data points that change over time.

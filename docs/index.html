Graph Functions

matrix_from_tables

Create a DMP (Dynamic MultiPartite) graph from one or more dataframes.

Parameters:

  • tables: pandas.DataFrame or list of pandas.DataFrame
  • relationships: list of lists
  • dynamic_col: str or list of str
  • join_token: str

Returns:

  • A: scipy.sparse.csr_matrix
  • attributes: list of lists

create_edge_list

Create an edge list from tables, relationships, dynamic columns, and weight columns.

Parameters:

  • tables: pandas.DataFrame or list of pandas.DataFrame
  • relationships: list of lists

Returns:

  • edge_list: pandas.DataFrame

extract_node_time_info

Extract node and time information from an edge list.

Parameters:

  • edge_list: pandas.DataFrame
  • join_token: str

Returns:

  • nodes: list
  • partitions: list
  • times: list
